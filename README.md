# Weather App

This is a simple weather application that provides current weather information for any city. The app uses a weather API to fetch real-time weather data and displays the temperature, humidity, and weather conditions in an easy-to-understand interface.

## Overview
The Weather App allows users to check the weather of any city by simply entering the city name. The app displays key weather details such as temperature, humidity, weather description, and other relevant data fetched from a weather API.

## Features
- **Search Weather by City**: Enter a city name to get the current weather details.
- **Current Weather Information**: Displays temperature, humidity, weather description, and other weather-related data.
- **Responsive UI**: The app has a responsive design, ensuring it works well on both desktop and mobile devices.
- **Units Customization**: Choose between Celsius or Fahrenheit for temperature.
